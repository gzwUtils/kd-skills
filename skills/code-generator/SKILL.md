# 代码生成器

该技能自动为 Java Spring Boot 应用程序生成 CRUD 代码和基础架构，减少重复编码工作。

## 核心功能

### 实体生成
- **数据库实体**：根据数据库表结构生成 JPA/MyBatis 实体类
- **DTO**：创建用于 API 请求/响应的数据传输对象
- **VO/BO**：生成视图对象和业务对象

### CRUD 操作
- **数据访问层**：生成 MyBatis-Plus 或 JPA 数据访问层
- **服务层**：创建包含业务逻辑的服务层
- **控制器层**：生成带有标准接口端点的 REST 控制器
- **Mapper XML**：用于基于 MyBatis 的项目

### 项目结构
- **包组织**：遵循标准 Spring Boot 项目结构
- **配置文件**：生成 application.properties/yml 配置文件
- **安全配置**：基础安全配置
- **日志配置**：标准日志配置

### 虚拟电厂专项
- **设备管理**：生成智能设备实体相关代码
- **能源交易**：生成能源交易实体相关代码
- **监控数据**：生成实时监控实体相关代码
- **用户管理**：生成平台用户相关代码

## 使用示例

```bash
# 为 Device 实体生成代码
# 输入：数据库表结构或实体定义
# 输出：实体类、DTO、Service、Controller、Repository
```

## 集成 MyBatis-Plus

- **代码生成器**：使用 MyBatis-Plus 代码生成器自动创建代码
- **CRUD 方法**：利用 MyBatis-Plus 内置方法
- **查询构造器**：生成包含查询构造器示例的代码
- **分页支持**：包含分页功能支持

## 最佳实践

- **命名规范**：遵循 Java 和 Spring Boot 命名标准
- **代码结构**：保持代码结构清晰有序
- **异常处理**：在生成的代码中包含基础异常处理
- **输入校验**：添加输入校验注解
- **文档注释**：包含 Javadoc 注释
